# Piece graphics

Set: **Cburnett** (by Colin M.L. Burnett), Lichess’ default piece set.

Source: Wikimedia Commons (rendered from the `Chess_*45.svg` files).
License: GPLv2+ / BSD / GFDL (multi-license, free to use with attribution).

Files: `wP/wN/wB/wR/wQ/wK` (White) and `bP/bN/bB/bR/bQ/bK` (Black), each a PNG
with transparency. The board loads them from this folder; if they are missing,
it automatically falls back to Unicode pieces.
